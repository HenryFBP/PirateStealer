# StanGrabber
Official StanGrabber repos

# Features
- Discord Credit Card Stealing
    - Credit Card Number
    - CVC
    - Credit Card Expirations (month & years)
    - Country
    - State
    - City
    - ZIP Code
    - Address
- Discord Login Stealing
    - Email
    - Password
- Discord Change Password
    - Old Password
    - New Password
- Discord Change Email
    - New Email
    - Password
